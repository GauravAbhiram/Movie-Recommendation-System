## Movie Recommendation System

This project implements a **content-based movie recommendation system** using **Machine Learning and Natural Language Processing techniques**. The system recommends movies similar to a given movie based on textual features such as genres and movie descriptions.

##  Project Overview

With the growing number of movies available on digital platforms, it becomes difficult for users to find movies that match their interests. This project aims to solve this problem by developing a recommendation system that analyzes movie content and suggests similar movies.

The system uses **TF-IDF vectorization** to convert textual data into numerical features and **cosine similarity** to measure similarity between movies.

##  Machine Learning Techniques

- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Filtering


##  Project Workflow

1. Load movie dataset
2. Data preprocessing and feature selection
3. Combine movie genres and descriptions
4. Convert text into numerical vectors using TF-IDF
5. Compute cosine similarity between movies
6. Recommend top-N similar movies
7. Visualize dataset insights using graphs
